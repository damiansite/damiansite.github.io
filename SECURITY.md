# Security Policy

## Supported Versions

The website is a personal project without help or additional resources from other people. Please note its mainly developed for PCs, but it can run fairly well on mobile devices. The tablets sadly aren't taken in consideration and were not tasted to ensure quality.

## Reporting a Vulnerability

There are no vulnerabilities known till now. In case someone finds any, would be amazing if he/she/anyone could provide some insight. Anticipated thanks!
